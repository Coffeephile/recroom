Installation

# Install using npm (comes with Node.js)
# recroom is installed globally so you can use it on the command
# line from any directory.
npm install -g recroom

recroom requires Node.js. Install node to get started with recroom.

recroom uses Ember.js as the framework to build your app’s controllers, models, and views. Our documentation will walk you through the basics of how to use Ember.js to build your app, but you can always refer to the Ember.js docs if you need aditional information.

If you get an error about permissions and are on Linux, Mac OS X, or another flavour of Unix, you may need to use sudo:
sudo npm install -g recroom.

Create Your First Project

# Create a project (you can call it whatever you want).
# A new folder will be created inside the directory you
# run recroom in.
recroom new my-app

Think up a name for your project and run recroom new my-app.

If you don’t name your project, recroom will create a new project named recroom-app. You can change the name of your project later by hand.

Start Your App

# Enter your project's folder.
# TODO: cd into the project on creation automatically?
cd my-app

# Run your recroom app.
recroom run

Now that you’ve created an app, you can start it by running recroom run. develop it and test it. When you server your app, recroom will automatically compile your JavaScript, CoffeeScript, and Stylus.

Whenever you type recroom run, your app will automatically be opened in your system’s default web browser.

If you’d like to run your app via the recroom server, without opening your browser, type recroom serve instead.
